# Portfolio Bintang

Project ini adalah website portfolio pribadi yang dibuat untuk memenuhi tugas Pemrograman Dasar. Website ini menggunakan HTML5 dan CSS3 murni tanpa framework eksternal.

## Struktur Folder

```
portfolio/
├── index.html   # Halaman utama
├── style.css    # File styling CSS
└── README.md    # Dokumentasi ini
```

## Fitur Utama

- **Header & Navigasi**: Menu navigasi yang sticky untuk kemudahan akses.
- **About Section**: Informasi diri dan foto profil.
- **Skills Section**: Daftar keahlian yang ditampilkan dalam bentuk kartu grid.
- **Portfolio Section**: Showcase proyek-proyek terbaru.
- **Contact Section**: Formulir kontak dan informasi sosial media.
- **Responsive Design**: Tampilan menyesuaikan ukuran layar (Desktop & Mobile).

## Persyaratan CSS yang Terpenuhi

1. **Selektor Elemen**: Styling langsung pada tag HTML (cth: `body`, `h1`, `p`).
2. **Selektor ID**: Lebih dari 3 ID digunakan (cth: `#main-header`, `#navbar`, `#about`, `#skills`, `#portfolio`).
3. **Selektor Kelas**: Lebih dari 5 kelas digunakan (cth: `.container`, `.btn`, `.card`, `.section-title`, `.text-center`).
4. **Selektor Universal**: Reset margin/padding menggunakan `*`.
5. **Pengelompokan Selektor**: Styling bersama untuk elemen serupa (cth: `h1, h2, h3` dan `input, textarea`).

## Cara Menjalankan

Cukup buka file `index.html` di web browser modern (Chrome, Firefox, Edge).
